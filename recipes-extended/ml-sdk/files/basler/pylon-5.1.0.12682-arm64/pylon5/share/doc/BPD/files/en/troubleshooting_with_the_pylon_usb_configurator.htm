Troubleshooting with the pylon USB Configurator

The pylon USB Configurator is a useful tool if you're experiencing problems with your cameras. It can detect error conditions and offers solutions.

Error Conditions

Error conditions are shown in the device info pane. In the example below, an acA1300-30um camera is connected to a USB 2.0 adapter instead of an USB 3.0 adapter. In this case, the camera is detected but will not work. At the bottom of the pane, detailed information about the problem and suggestions how to resolve the error condition are shown.

Generating Support Information

The USB Configurator provides a convenient way of generating a report about the current USB device tree. If you are experiencing problems and need assistance, you can send the report to Basler technical support. There, the report will greatly help to assess your current installation and give you detailed advice.

To generate support information

  1. In the pylon USB Configurator window:
    1. Click the question mark ? in the menu bar.
    2. Click Generate Support Information... in the dropdown menu.
    The Support Information window opens displaying the report.
  2. You can copy the report to the clipboard or save it:
    • If you want to send it to Basler technical support, click the Copy to Clipboard button and paste the report into an email message to Basler technical support.
    • Click the Save button to save the report on your PC.
