Field Output Mode

The Field Output Mode camera feature allows you to control the way in which image data is transmitted on acA750-30gm/gc cameras.

Using the Feature

Choosing a Field Output Mode

To choose a field output mode, set the FieldOutputMode parameter to one of the following values:

  • Field0
  • Field1
  • ConcatenatedNewFields
  • DeinterlacedNewFields

How It Works

Due to the architecture of the sensor used in the acA750-30gm/gc camera models, there is only one vertical shift register for every two physical pixels in the sensor. This leads to a "field" readout scheme for the sensor. There are two fields that can be read out of the sensor: "Field 0" and "Field 1".

The main difference between Field 0 and Field 1 is that they combine the pixels in the sensor rows in different ways.

Field 0 Readout

To read out Field 0 pixel data, the camera combines the pixel values from row 0 with the pixel values from row 1, the pixel values from row 2 with the pixel values from row 3, and so on.

Field 1 Readout

To read out Field 1 pixel data, the camera combines the pixel values from row 1 with the pixel values from row 2, the pixel values from row 3 with the pixel values from row 4, and so on.

Available Field Output Modes

On acA750-30gm/gc cameras, four field output modes are available.

Field 0 Output Mode

In Field 0 output mode, the camera transmits pixel data from field 0 only. Every time the camera receives a Frame Start trigger signal, it acquires an image using the field 0 readout scheme. Because pairs of rows are combined, the height of the image is halved.

Field 1 Output Mode

In Field 1 output mode, the camera transmits pixel data from field 1 only. Every time the camera receives a Frame Start trigger signal, it acquires an image using the field 1 readout scheme. Because pairs of rows are combined, the height of the image is halved.

Concatenated New Fields Output Mode

In Concatenated New Fields output mode, the camera transmits concatenated pixel data from field 0 and field 1. Every time the camera receives a Frame Start trigger signal, it acquires one image using the field 0 readout scheme and one image using the field 1 readout scheme. The data from those two images is concatenated and the concatenated image data is transmitted as a single image.

If a camera is operating in Concatenated New Fields output mode with an acquisition rate of, e.g., 30 frames per second, it acquires 60 fields per second. Since two fields are combined to produce one image, the camera ends up transmitting 30 frames per second.

Unlike the Field 0 and Field 1 output modes, the Concatenated New Fields provides pixel data for a "full height" image.

  • When using the Concatenated New Fields output mode, the image data must be deinterlaced in order to use it effectively.
  • If the target object is moving, there can be significant distortion in the transmitted images.

Deinterlaced New Fields Output Mode

In Deinterlaced New Fields output mode, the camera transmits deinterlaced pixel data from field 0 and field 1. Every time the camera receives a Frame Start trigger signal, it acquires one image using the field 0 readout scheme and one image using the field 1 readout scheme. The data from those two images is deinterlaced and the deinterlaced image data is transmitted as a single image.

If a camera is operating in Deinterlaced New Fields output mode with an acquisition rate of, e.g., 30 frames per second, it acquires 60 fields per second. Since two fields are combined to produce one image, the camera ends up transmitting 30 frames per second.

Unlike the Field 0 and Field 1 output modes, the Deinterlaced New Fields provides pixel data for a "full height" image.

If the target object is moving, there can be significant distortion in the transmitted images.

Sample Code

  • C++ (Native)
  • C++ (Generic)
  • C#
  • C.NET
  • C
// Set the field output mode to Field 0
camera.FieldOutputMode.SetValue(FieldOutputMode_Field0);
// Set the field output mode to Field 1
camera.FieldOutputMode.SetValue(FieldOutputMode_Field0);
// Set the field output mode to Concatenated New Fields
camera.FieldOutputMode.SetValue(FieldOutputMode_ConcatenatedNewFields);
// Set the field output mode to Deinterlaced New Fields
camera.FieldOutputMode.SetValue(FieldOutputMode_DeinterlacedNewFields);
INodeMap& nodemap = camera.GetNodeMap();
// Set the field output mode to Field 0
CEnumerationPtr(nodemap.GetNode("FieldOutputMode"))->FromString("Field0");
// Set the field output mode to Field 1
CEnumerationPtr(nodemap.GetNode("FieldOutputMode"))->FromString("Field0");
// Set the field output mode to Concatenated New Fields
CEnumerationPtr(nodemap.GetNode("FieldOutputMode"))->FromString("ConcatenatedNewFields");
// Set the field output mode to Deinterlaced New Fields
CEnumerationPtr(nodemap.GetNode("FieldOutputMode"))->FromString("DeinterlacedNewFields");
// Set the field output mode to Field 0
camera.Parameters[PLCamera.FieldOutputMode].SetValue(PLCamera.FieldOutputMode.Field0);
// Set the field output mode to Field 1
camera.Parameters[PLCamera.FieldOutputMode].SetValue(PLCamera.FieldOutputMode.Field0);
// Set the field output mode to Concatenated New Fields
camera.Parameters[PLCamera.FieldOutputMode].SetValue(PLCamera.FieldOutputMode.ConcatenatedNewFields);
// Set the field output mode to Deinterlaced New Fields
camera.Parameters[PLCamera.FieldOutputMode].SetValue(PLCamera.FieldOutputMode.DeinterlacedNewFields);
// Set the field output mode to Field 0
Pylon.DeviceFeatureFromString(hdev, "FieldOutputMode", "Field0");
// Set the field output mode to Field 1
Pylon.DeviceFeatureFromString(hdev, "FieldOutputMode", "Field0");
// Set the field output mode to Concatenated New Fields
Pylon.DeviceFeatureFromString(hdev, "FieldOutputMode", "ConcatenatedNewFields");
// Set the field output mode to Deinterlaced New Fields
Pylon.DeviceFeatureFromString(hdev, "FieldOutputMode", "DeinterlacedNewFields");
/* Macro to check for errors */
#define CHECK(errc) if (GENAPI_E_OK != errc) printErrorAndExit(errc)
GENAPIC_RESULT errRes = GENAPI_E_OK;  /* Return value of pylon methods */
/* Set the field output mode to Field 0 */
errRes = PylonDeviceFeatureFromString(hdev, "FieldOutputMode", "Field0");
CHECK(errRes);
/* Set the field output mode to Field 1 */
errRes = PylonDeviceFeatureFromString(hdev, "FieldOutputMode", "Field0");
CHECK(errRes);
/* Set the field output mode to Concatenated New Fields */
errRes = PylonDeviceFeatureFromString(hdev, "FieldOutputMode", "ConcatenatedNewFields");
CHECK(errRes);
/* Set the field output mode to Deinterlaced New Fields */
errRes = PylonDeviceFeatureFromString(hdev, "FieldOutputMode", "DeinterlacedNewFields");
CHECK(errRes);
