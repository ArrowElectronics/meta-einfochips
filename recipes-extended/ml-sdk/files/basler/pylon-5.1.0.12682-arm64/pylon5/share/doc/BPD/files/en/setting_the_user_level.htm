Setting the User Level

Depending on the user level set in pylon Viewer, only a subset of camera settings will be displayed.

Available User Levels

  • Beginner: The Features pane will only display the most basic camera settings.
  • Expert: The Features pane will display the most commonly used camera settings.  
  • Guru: The Features pane will display all camera features, including the most advanced ones.
