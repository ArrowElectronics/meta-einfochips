Hardware Installation (dart BCON for LVDS)

This topic tells you how to install a Basler dart BCON for LVDS camera module.

Prerequisites

The pylon Camera Software Suite for Linux must be installed.

Installation

To install the camera module:

  1. Connect the flexible flat cable with the camera module and the image processing system.
    To insert the cable:
    1. Push the end of the cable firmly into the connector with the ground tab facing down until the retaining tabs slide into the recesses on both sides.
    2. Flip the locking bar down to lock the connector.
  2. Turn on the power supply.
    The camera's indicator LED lights up.
  3. Check the power consumption of the camera module. If it needs more than 1.5 W, turn the power off immediately and check for short circuits and miswirings.
