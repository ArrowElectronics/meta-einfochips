What's New?

This version of the Basler Product Documentation includes documentation for the new pylon Viewer features. It also includes minor enhancements.

New pylon Viewer Features

The pylon Viewer section has been updated to include features that have been added in the latest release of the pylon Camera Software Suite, e.g., the Histogram feature, the Crosshair and Grid feature, or the Color Calibrator feature.

Camera Emulation Devices

The pylon Camera Software Suite offers camera emulation devices. These devices allow you to try out basic camera features and to develop applications even if no physical camera is available.

Documentation for these devices is now available in the Camera Emulation topic. You can also access this topic directly from the pylon Viewer. Just press F1 while using an emulation device.
