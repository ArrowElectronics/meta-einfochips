Overlapping Image Acquisition

With overlapping image acquisition, the exposure of a new image begins while the camera is still reading out the sensor data of the previous image.

This is useful if you want to increase the frame rate of your camera.

Prerequisites

  • The camera's shutter mode must be set to Global or Rolling. Overlapping image acquisition is not possible in Global Reset Release shutter mode.
  • On some camera models, overlapping image acquisition is only possible when the camera is configured for free run image acquisition.

Enabling Overlapping Image Acquisition

Automatic

On most cameras, there is no camera parameter that allows you to turn overlapping image acquisition on or off. Instead, the way that you operate the camera determines whether image acquisitions are overlapped.

Image acquisitions are overlapped if the following condition applies:

Frame Period â‰¤ Exposure Time + Sensor Readout Time

  • Frame Period: Period of time from the start of one image acquisition to the start of the next image acquisition.
  • Exposure Time:
    • Timed exposure mode: Equals the ExposureTimeAbsExposureTime parameter value.
    • Trigger Width exposure mode: Equals the width of the hardware trigger signal.
  • Sensor Readout Time: Time it takes to read out the data of an image from the sensor. If available, you can use the Sensor Readout Time feature to get the value.

If the camera is configured for free run image acquisition, overlapping image acquisitions occur automatically.

If the camera is configured for triggered image acquisition, overlapping image acquisition occur whenever you trigger the camera at a sufficiently high rate.

Manual

On some camera models, you can manually enable or disable overlapping image acquisition. To do so, set the OverlapMode parameter to one of the following values:

  • On: Enables overlapping image acquisition. The camera automatically overlaps exposures and readouts.
  • Off: Disables overlapping image acquisition. The camera never overlaps exposures and readouts. This can improve image quality, especially when you are operating the camera at low frame rates.

Avoiding Overtriggering

Global Shutter Mode

If you have configured the Global shutter mode and the Trigger Width exposure mode on the camera and you want to use overlapping image acquisition, the following guideline applies:

You must not stop the exposure of the current frame until readout of the previously acquired frame is complete.

If you violate this guideline, the camera drops the current frame and declares a Frame Start Overtrigger event.

You can avoid violating this guideline by monitoring the cameraâ€™s Frame Trigger Wait signal and by specifying an appropriate value for the Exposure Overlap Time Max parameter.
